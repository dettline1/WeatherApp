## 🎥 Демонстрация функций

### 1️⃣ Поиск погоды

**Шаги:**
1. Введите название города (например, "Moscow")
2. Нажмите "Показать погоду" или Enter
3. Получите актуальную информацию о погоде

**Результат:**
- Температура: +5.2°C
- Ощущается как: +2.1°C
- Влажность: 76%
- Скорость ветра: 4.5 м/с
- Описание: "Ясно"
- Иконка погоды

### 2️⃣ Автоопределение города

**Шаги:**
1. Нажмите кнопку "📍 Определить мой город"
2. Приложение определит ваш город по IP
3. Автоматически загрузится погода

**Как работает:**
- Запрос к ipinfo.io API
- Получение города по IP
- Автоматический поиск погоды

### 3️⃣ История запросов

**Особенности:**
- Сохраняются все запросы в SQLite
- Отображаются последние 5 записей
- Клик по записи → повторный поиск
- Хранение: город, страна, температура, время

### 4️⃣ Смена языка

**Языки:**
- 🇷🇺 Русский (по умолчанию)
- 🇬🇧 English

**Что переводится:**
- Весь интерфейс
- Описания погоды от API
- Сообщения об ошибках

### 5️⃣ Обработка ошибок

**Типы ошибок:**

```

│  ⚠️ Город не найден. Проверьте название.   │
└─────────────────────────────────────────────┘
```


## 🔍 Примеры использования

### Пример 1: Поиск погоды в Москве

**Запрос:**
```
City: Moscow
```

**Ответ:**
```json
{
  "city": "Moscow",
  "country": "RU",
  "temperature": 5.2,
  "feels_like": 2.1,
  "humidity": 76,
  "wind_speed": 4.5,
  "description": "Ясно",
  "icon": "01d"
}
```

### Пример 2: Поиск на английском

**Запрос:**
```
City: New York
Language: EN
```

**Ответ:**
```json
{
  "city": "New York",
  "country": "US",
  "temperature": 8.5,
  "feels_like": 6.2,
  "humidity": 65,
  "wind_speed": 3.8,
  "description": "Clear sky",
  "icon": "01d"
}
```

### Пример 3: Ошибка - город не найден

**Запрос:**
```
City: Nonexistent City
```

**Ответ:**
```json
{
  "error": "error_city_not_found"
}
```

**Отображение:**
```
⚠️ Город не найден. Проверьте название.
```
